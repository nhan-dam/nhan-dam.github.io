## Introduction and Scope

This course is written for a machine learning (ML) practitioner with a solid background in supervised learning, neural network training, and standard deep learning tooling, but with no prior exposure to reinforcement learning (RL). The goal is threefold: to build sufficient theoretical understanding to reason about why RL and reinforcement learning from human feedback (RLHF) behave as they do; to develop the debugging intuitions needed to diagnose training failures; and to produce working, non-trivial implementations on the Mac workstation.

The course is divided into three phases. Phase 1 covers the RL foundations that underpin RLHF. Phase 2 covers the classical three-step RLHF pipeline. Phase 3 covers the modern alternatives that have displaced proximal policy optimisation (PPO)-based RLHF in most practical settings. Each module contains a theory section, a set of design intuitions aimed specifically at debugging and training stability, and a hands-on project with concrete implementation guidance for the local hardware.
