# RLHF Curriculum

A self-directed study programme working from foundational RL theory up to modern RLHF methods used in production language models.

---

## 1. Motivation

Reinforcement learning from human feedback (RLHF) is the dominant technique for aligning large language models. Understanding it properly requires building up from first principles: Markov decision processes, dynamic programming, policy gradient methods, and then the specific adaptations (reward modelling, PPO fine-tuning, DPO) used in practice.
